# Cenari Management System

## Synopsis
A content management system.

## Installation
Prerequisites:
* MySQL Server (mariadb)
* PHP 7+
* PHP - php_mbstring
* PHP - php_pdo_mysql
* PHP - php_sockets
* Web Server (apache|nginx)
