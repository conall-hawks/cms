CSS Pagination

This pagination method works by exploiting the input:checked pseudo-selector. When you click on one of the "links" to the left you are actually clicking on an <input>'s accompanying <label>, which sets the <input> to :checked, thus applying a style to a sibling-element (using input:checked + sibling). The result emulates a "page-change" without having to reload the page. Take a look at the code to discover more about how it works!
